# Content Tracking Guide

**Purpose**: Guide for tracking Tech Teardown projects from planning through publication.

---

## Overview

### Why Track Content?
- **Progress**: Know where you are in each project
- **Planning**: Plan future projects effectively
- **Quality**: Ensure consistent quality
- **Learning**: Learn from each project
- **Efficiency**: Identify bottlenecks and improve

### What to Track
- **Project Status**: Current phase of each project
- **Progress**: Completion percentage
- **Timeline**: Start dates, deadlines, actual completion
- **Metrics**: Time, cost, output metrics
- **Lessons**: What worked, what didn't

---

## Project Status Tracking

### Status Levels

#### Planning
- **Definition**: Project is being planned
- **Activities**: Research, device selection, planning
- **Deliverables**: Research documents, plan

#### In Progress
- **Definition**: Project is actively being worked on
- **Activities**: Production, post-production
- **Deliverables**: Footage, audio, edits

#### Completed
- **Definition**: Project is finished and published
- **Activities**: Publication, promotion
- **Deliverables**: Published video, metrics

#### Archived
- **Definition**: Project is archived
- **Activities**: None (archived)
- **Deliverables**: Archived materials

### Status Tracking Method
- **Project Metadata**: Update status in project metadata file
- **Master Tracker**: Maintain master project list
- **Regular Updates**: Update status regularly

## Progress Tracking

### Progress Phases

#### Phase 1: Planning (0-20%)
- [ ] Device selected
- [ ] Research started
- [ ] Script outline complete
- [ ] Equipment prepared
- [ ] **Progress**: 20%

#### Phase 2: Research (20-40%)
- [ ] Device research complete
- [ ] History research complete
- [ ] Script written
- [ ] **Progress**: 40%

#### Phase 3: Production (40-70%)
- [ ] Device acquired
- [ ] Footage recorded
- [ ] Audio recorded
- [ ] **Progress**: 70%

#### Phase 4: Post-Production (70-95%)
- [ ] Video edited
- [ ] Audio mixed
- [ ] Graphics added
- [ ] Final export complete
- [ ] **Progress**: 95%

#### Phase 5: Publication (95-100%)
- [ ] Video published
- [ ] Promoted
- [ ] Metrics collected
- [ ] **Progress**: 100%

### Progress Tracking Method
- **Checklist**: Use phase checklists
- **Percentage**: Track percentage complete
- **Regular Updates**: Update progress regularly

## Common Mistakes to Avoid

### ❌ Not Tracking
- **Problem**: No visibility into progress
- **Solution**: Set up tracking system, track regularly

### ❌ Over-Tracking
- **Problem**: Too much time spent tracking
- **Solution**: Keep tracking simple, focus on key metrics

### ❌ Not Using Data
- **Problem**: Collecting data but not using it
- **Solution**: Review data regularly, make adjustments

### ❌ Inconsistent Tracking
- **Problem**: Inconsistent or missing data
- **Solution**: Establish routine, track consistently
