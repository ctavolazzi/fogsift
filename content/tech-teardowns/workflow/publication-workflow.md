# Publication Workflow

**Purpose**: Step-by-step workflow for publishing Tech Teardown videos.

---

## Pre-Publication Checklist

### Video Quality
- [ ] Video is complete and final
- [ ] Audio is synced and clear
- [ ] Color correction is consistent
- [ ] Graphics/text are correct
- [ ] No mistakes or errors
- [ ] Professional quality

### Content Quality
- [ ] Facts are accurate
- [ ] Sources are cited
- [ ] Script is polished
- [ ] Story is compelling
- [ ] Length is appropriate

### Technical Quality
- [ ] Resolution is correct (1080p minimum)
- [ ] Frame rate is correct (30fps)
- [ ] Audio levels are correct
- [ ] File format is correct (MP4)
- [ ] File size is reasonable

---

## Publication Steps

### Step 1: Final Export

#### Export Settings
- **Resolution**: 1080p (1920x1080) or 4K
- **Frame Rate**: 30fps
- **Codec**: H.264
- **Bitrate**: 10-20 Mbps (1080p)
- **Audio**: AAC, 128-192 kbps

#### Export Process
1. Review final edit
2. Set export settings
3. Export video
4. Quality check export
5. Save to `06-exports/final/`

### Step 2: Thumbnail Creation

#### Thumbnail Requirements
- **Size**: 1280x720 pixels
- **Format**: JPG or PNG
- **File Size**: Under 2MB
- **Quality**: Clear, readable text

#### Thumbnail Design
- **Device Image**: Clear image of device
- **Text**: Readable title/text
- **Branding**: Consistent branding (if applicable)
- **Engaging**: Eye-catching, makes viewer want to click

#### Thumbnail Creation
1. Create thumbnail design
2. Export at correct size
3. Save to `04-graphics/thumbnails/`
4. Create multiple versions
5. Test which works best

### Step 3: Video Description

#### Description Structure
- **Hook**: Engaging first line
- **Summary**: Brief video summary
- **Key Points**: Bullet points of key topics
- **Timestamps**: Timestamps for sections (if applicable)
- **Links**: Links to related content
- **Tags**: Relevant tags/keywords

#### Description Template
```
[Engaging hook that makes viewer want to watch]

In this video, we tear apart [Device Name] and explore [key story points].

Key topics covered:
• [Topic 1]
• [Topic 2]
• [Topic 3]

Timestamps:
0:00 - Introduction
2:00 - Historical Context
5:00 - The Politics & Drama
10:00 - The Teardown
15:00 - Significance & Impact
18:00 - Reassembly Attempt
19:30 - Conclusion

Related videos:
• [Link to related video 1]
• [Link to related video 2]

Sources:
• [Source 1]
• [Source 2]

#TechTeardown #VintageTech #DeviceName
```

### Step 4: Tags and Metadata

#### Tags
- **Primary Tags**: Main topics (e.g., #TechTeardown, #VintageTech)
- **Device Tags**: Device-specific tags
- **Topic Tags**: Topic-specific tags
- **Platform Tags**: Platform-specific tags (if applicable)

#### Metadata
- **Title**: Clear, descriptive title
- **Category**: Appropriate category
- **Language**: Language of video
- **Visibility**: Public, Unlisted, or Private

### Step 5: Upload

#### Upload Process
1. Log into platform
2. Click upload/create
3. Select video file
4. Enter title
5. Add description
6. Add tags
7. Upload thumbnail
8. Set metadata
9. Set visibility
10. Schedule or publish

#### Upload Settings
- **Privacy**: Public (or Unlisted for review)
- **Category**: Technology/Education
- **Language**: English (or appropriate)
- **Comments**: Enable comments
- **Subtitles**: Add subtitles if available

### Step 6: Review

#### Review Process
1. Watch uploaded video
2. Check quality
3. Verify description
4. Check tags
5. Test thumbnail
6. Verify links work
7. Make any corrections

#### If Issues Found
- **Re-upload**: If major issues, re-upload
- **Edit**: If minor issues, edit on platform
- **Note**: Note issues for future projects

### Step 7: Publish

#### Publish Options
- **Publish Now**: Publish immediately
- **Schedule**: Schedule for specific time
- **Unlisted**: Publish as unlisted for review first

#### After Publishing
1. Share on social media
2. Update project tracker
3. Monitor engagement
4. Respond to comments
5. Track metrics
