## Feature Development Workflow

1. **Create branch** from `main`: `feature/<feature-name>`
2. **Create spec document** in this directory: `02.XX_YYYYMMDD_feature_name.md`
3. **Create implementation checklist**: `02.XX_IMPLEMENTATION_CHECKLIST.md`
4. **Implement feature** following the checklist
5. **Test** across browsers and devices
6. **Create PR** with before/after screenshots
7. **Update index** when merged
